void UserInterface::inputFirstCode ( )

初期位置コードを入力する。 whileループに入っており、ENTERボタンを押すまでループを抜けることはない。 UPボタンとDOWNボタンで数字の大小を選択し、RightボタンとLeftボタンで変更する数字の桁数を選択する。
